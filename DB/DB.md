# DataBase란?
- DataBase는 체계적인 데이터 컬렉션을 저장, 관리 및 보호하기 위한 디지털 저장소입니다
- 단어, 숫자, 이미지, 비디오 및 파일을 포함한 모든 유형의 데이터가 포함될 수 있습니다
- DBMS라는 소프트웨어를 사용하여 데이터를 저장 검색 및 편집할 수 있습니다.
### 데이터 베이스가 중요한 이유
- **효율적인 확장**: 대량의 데이터를 관리할 수 있습니다
- **데이터 무결성**: 데이터 일관성을 유지하기 위한 규칙 및 조건이 있습니다
- **데이터 보안**: 모든 데이터와 관련된 개인정보 보호 및 규정 준수 요구사항을 지원합니다
- **데이터 분석**: 소프트웨어 시슽템은 데이터베이스를 사용하여 데이터를 분석합니다 시스템의 추세와 패턴을 식별하거나 예측할 수 있습니다

### 데이터베이스에서 모델이란?
데이터베이스의 논리적 구조를 보여줍니다
데이터 저장, 구성, 조작할 수 있는 방법을 결정하는 관계와 규칙을 정의합니다

### 데이터 베이스 발전 순서
1. 계층적 데이터베이스
2. 네트워크 데이터베이스
3. 관계형 데이터베이스
4. 객체 지향 데이터베이스
5. NoSQL 데이터베이스
