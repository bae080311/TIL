# Blocking & Non-Blocking

- 블로킹과 논 블로킹은 다른 작업을 처리하기 위해 현재 작업을 block(차단 또는 대기)을 하는지 마는지 유무를 나타내는 프로세스의 실행 방식입니다
- 동기와 비동기가 전체적인 작업에 대한 순차적인 흐름 유무라고 치면 블로킹과 논블로킹은 전체적인 작업의 흐름 자체를 막냐 안막냐로 볼 수 있습니다
- 예를 들자면 블로킹은 아이스크림을 동시에 한 개만 먹을 수 있고 아이스크림을 한 개 다 먹을 때까지 대기해야 하지만 논블로킹은 한 개의 아이스크림을 먹으면서 다른 아이스크림도 먹을 수 있습니다

### 비동기와 논블로킹의 차

비동기는 작업의 완료 시점에 초점이 있고 논블로킹은 함수 호출시 대기 여부에 초점이 있다고 합니다
둘은 자주 함께 사용되지만 같은 개념이 아닙니다
