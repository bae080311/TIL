# EC2란?

EC2는 **Amazon EC2 (Elastic Compute Cloud)**의 줄임말로,
AWS에서 제공하는 가상 서버 서비스입니다.

- Virtual Machine
- 재구성이 가능한 컴퓨팅 리소스
- 쉽게 확장/축소되는 컴퓨터 용량
- 고객 업무 영역에 따른 다양한 인스턴스 타입 제공
- 사용한 만큼만 과금 (pay-as-you-go)

### 인스턴스

- 범용 (M4): 컴퓨팅, 메모리 네트워크 리소스의 균형적 사용
- 컴퓨팅 (C4): EC2에서 최고 성능의 프로세서, 성능 대비 저렴한 가격
- 메모리 (R4): 메모리 용량이 많이 필요한 애플리케이션용
- 선응 순간 확장 (T2): 성능 순간 확장 가능, 기본 제공, 가장 많이 사용 됨
- GPU (G3, P2): 그래픽 및 일반 목적의 GPU 컴퓨팅 애플리케이션
- 높은 I/O (I2): SSD 기반의 초고속 인스턴스 스토리지
- 고밀도 스토리지 (D2): 높은 디스크 처리량, 단위당 최소의 가격

### 과금 옵션 및 비용 최적화

- On-Demand: 약정없이 쓴 만큼만 지불 -> 갑작스런 트래픽이나 예측하기 어려운 경우 신규 서비스
- Reserved: 1년 혹은 3년 약정 40~70% 할인 -> 항상 사용 중인 안정화된 서버 자원을 위한 요금제
- Spot: 남은 자원에 대한 경매 방식 더 높은 가격으로 입찰할 경우 바로 양도 될 수 있으나 80~90% 저렴 -> 단기적으로 수요가 많을 때 유리
